# Complexity Anaylsis
- N is number of host
- w/ flow Priority: 
    - Time Complexity: O(NlongN) from sorting
    - Space Complexity: O(1)

- w/o flow priority:
    - 每個訂閱都直接綁定unique port, 每個packet根據subscriber的數量複製一次
    - Space: O(N)
        + argument: 看似可以接受，但如果訂閱者很多就會把bandwith擠爆
    - Time Complexity: O(N)
